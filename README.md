# Projects of GenAI

A collection of generative AI projects leveraging the power of [OpenAI] [LangChain] and [LangGraph]. This repository showcases practical applications and experiments in natural language processing, conversational agents, and workflow orchestration using state-of-the-art tools.

---

## 🚀 Overview

This repository contains multiple projects and demos that illustrate how to build intelligent applications with:

- **OpenAI**: Access powerful language models for text generation, summarization, Q&A, and more.
- **LangChain**: Chain together LLMs, tools, and data sources to create complex, multi-step workflows.
- **LangGraph**: Build and visualize dynamic, graph-based workflows for LLM-powered applications.

---

## 🧩 Features

- **Conversational Agents**: Chatbots and assistants powered by OpenAI GPT models.
- **Document QA**: Retrieve and answer questions from custom documents using LangChain.
- **Workflow Automation**: Orchestrate multi-step processes with LangGraph.
- **Custom Integrations**: Connect to APIs, databases, and external tools.
